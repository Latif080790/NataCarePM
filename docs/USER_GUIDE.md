## Pengenalan

### Apa itu NataCarePM?

NataCarePM adalah sistem manajemen proyek konstruksi yang membantu Anda:
- **Mengelola proyek** dari awal hingga selesai
- **Melacak keuangan** (anggaran, transaksi, cashflow)
- **Menghasilkan laporan** otomatis dengan AI
- **Menyimpan dokumen** secara aman di cloud
- **Kolaborasi tim** real-time

### Kebutuhan Sistem

- **Browser:** Chrome 90+, Firefox 88+, Safari 13+, Edge 90+
- **Koneksi Internet:** Minimum 1 Mbps (rekomendasi 5 Mbps)
- **Resolusi Layar:** Minimum 1280x720 (mobile: 375x667)

## Login & Akses Pertama

### 1. Membuka Aplikasi

1. Buka browser dan akses: `https://natacarepm.web.app`
2. Anda akan melihat halaman login

### 2. Login

1. Masukkan **Email** dan **Password** yang diberikan oleh administrator
2. Klik tombol **"Login"**
3. Jika berhasil, Anda akan diarahkan ke Dashboard

**Catatan:** Jika ini login pertama kali, Anda akan diminta untuk:
- Verifikasi email (cek inbox/spam)
- Mengubah password default
- Setup Two-Factor Authentication (2FA) - opsional tapi direkomendasikan

### 3. Lupa Password

1. Klik link **"Forgot Password?"** di halaman login
2. Masukkan email Anda
3. Cek email untuk link reset password
4. Klik link dan buat password baru
5. Login dengan password baru

### 4. Two-Factor Authentication (2FA)

**Untuk keamanan ekstra:**

1. Setelah login, klik **Profile** (pojok kanan atas)
2. Pilih **"Security Settings"**
3. Aktifkan **"Two-Factor Authentication"**
4. Pilih metode:
   - **SMS** - Kode dikirim via SMS
   - **Authenticator App** - Gunakan Google Authenticator/Authy
5. Ikuti instruksi untuk setup
6. Login berikutnya akan meminta kode 2FA

## Dashboard

### Tampilan Utama

Setelah login, Anda akan melihat Dashboard dengan:

#### 1. Ringkasan Proyek
- **Total Proyek:** Jumlah proyek aktif
- **Proyek Selesai:** Progress keseluruhan
- **Proyek Delayed:** Proyek yang terlambat

#### 2. Ringkasan Keuangan
- **Total Anggaran:** Budget semua proyek
- **Pengeluaran:** Total yang sudah digunakan
- **Sisa Budget:** Anggaran tersisa

#### 3. Chart & Grafik
- **Budget vs Actual:** Perbandingan anggaran dengan realisasi
- **Cashflow Bulanan:** Aliran kas masuk/keluar
- **Progress Timeline:** Garis waktu proyek

#### 4. Aktivitas Terbaru
- Transaksi terbaru
- Dokumen yang di-upload
- Perubahan proyek

### Navigasi

**Sidebar (kiri):**
- **Dashboard** - Halaman utama
- **Projects** - Daftar semua proyek
- **Finance** - Keuangan & transaksi
- **Reports** - Laporan otomatis
- **Documents** - Penyimpanan file
- **AI Assistant** - Chat dengan AI

**Top Bar (atas):**
- **Search** - Cari proyek, transaksi, dokumen
- **Notifications** - Notifikasi penting

## Manajemen Proyek

### Membuat Proyek Baru

1. Klik **"Projects"** di sidebar
2. Klik tombol **"+ Create Project"**
3. Isi form:
   - **Nama Proyek:** Misal "Pembangunan Gedung A"
   - **Deskripsi:** Penjelasan singkat
   - **Tipe:** Pilih jenis proyek (Konstruksi, Renovasi, dll)
   - **Budget:** Total anggaran (Rp)
   - **Tanggal Mulai:** Kapan proyek dimulai
   - **Tanggal Selesai:** Target selesai
   - **Lokasi:** Alamat proyek
4. Klik **"Create"**
5. Proyek baru akan muncul di daftar

### Melihat Detail Proyek

1. Klik salah satu proyek di daftar
2. Anda akan melihat:
   - **Overview:** Ringkasan proyek
   - **RAB/AHSP:** Rencana Anggaran Biaya
   - **Timeline:** Gantt chart
   - **Transactions:** Transaksi keuangan
   - **Progress:** Progres fisik
   - **Documents:** File terkait proyek

### Mengedit Proyek

1. Buka detail proyek
2. Klik tombol **"Edit"** (pojok kanan atas)
3. Ubah informasi yang diperlukan
4. Klik **"Save"**

## Keuangan & Transaksi

### Menambah Transaksi

1. Klik **"Finance"** di sidebar
2. Klik tombol **"+ Add Transaction"**
3. Isi form:
   - **Deskripsi:** Misal "Pembayaran Material"
   - **Jumlah:** Rp 10.000.000
   - **Tipe:** 
     - **Income** - Pemasukan (terima uang)
     - **Expense** - Pengeluaran (keluar uang)
   - **Kategori:** Pilih kategori (Material, Upah, dll)
   - **Proyek:** Pilih proyek terkait
   - **Tanggal:** Tanggal transaksi
   - **Metode Pembayaran:** Cash, Transfer, dll
4. Upload bukti (opsional): Foto kwitansi/invoice
5. Klik **"Save"**

### Melihat Cashflow

1. Klik **"Finance"** ‚Üí **"Cashflow"**
2. Pilih periode (bulan/tahun)
3. Lihat grafik:
   - **Garis Hijau:** Pemasukan
   - **Garis Merah:** Pengeluaran
   - **Garis Biru:** Saldo
4. Tabel detail di bawah grafik

### Budget Tracking

1. Klik **"Finance"** ‚Üí **"Budget"**
2. Lihat setiap kategori:
   - **Allocated:** Anggaran yang dialokasikan
   - **Spent:** Sudah digunakan
   - **Remaining:** Sisa
   - **Progress Bar:** Visual progress
3. **üî¥ Merah:** Over budget
4. **üü° Kuning:** Mendekati limit (>80%)
5. **üü¢ Hijau:** Aman

### Purchase Order (PO)

1. Klik **"Finance"** ‚Üí **"Purchase Orders"**
2. Klik **"+ Create PO"**
3. Isi:
   - **Vendor:** Pilih/tambah vendor
   - **Items:** Daftar barang/jasa
   - **Quantity & Price:** Jumlah dan harga
   - **Total:** Otomatis dihitung
4. Klik **"Create"**
5. PO bisa di-print atau di-email ke vendor

## Laporan

### Menghasilkan Laporan

1. Klik **"Reports"** di sidebar
2. Pilih jenis laporan:
   - **Budget Summary:** Ringkasan anggaran
   - **Cashflow Report:** Laporan arus kas
   - **Progress Report:** Laporan progres
   - **Transaction History:** Riwayat transaksi
3. Atur filter:
   - **Date Range:** Pilih periode (bulan ini, kuartal, custom)
   - **Project:** Filter per proyek (opsional)
   - **Category:** Filter per kategori
4. Klik **"Generate Report"**
5. Laporan akan ditampilkan

### Export Laporan

**Format yang tersedia:**
- **PDF** - Untuk print/share
- **Excel (XLSX)** - Untuk analisis lebih lanjut
- **CSV** - Data mentah

**Cara export:**
1. Setelah laporan ditampilkan
2. Klik tombol **"Export"**
3. Pilih format (PDF/Excel/CSV)
4. File akan otomatis ter-download

### Laporan Otomatis dengan AI

1. Klik **"AI Assistant"** di sidebar
2. Ketik: *"Buatkan laporan keuangan bulan ini"*
3. AI akan menganalisis data dan menghasilkan laporan
4. Klik **"Download PDF"** untuk save laporan

## Dokumen

### Upload Dokumen

1. Klik **"Documents"** di sidebar
2. Klik **"+ Upload"**
3. Pilih file dari komputer (atau drag & drop)
4. **Supported formats:**
   - **Dokumen:** PDF, DOC, DOCX, XLS, XLSX
   - **Gambar:** JPG, PNG, WebP
   - **Lainnya:** ZIP, RAR
5. Tambahkan:
   - **Nama:** Nama file (otomatis dari filename)
   - **Kategori:** Contract, Invoice, Drawing, dll
   - **Proyek:** Terkait proyek mana
   - **Tags:** Keyword untuk mudah cari
6. Klik **"Upload"**

### Mencari Dokumen

**Cara 1: Search Bar**
1. Ketik nama file/keyword di search box
2. Hasil akan muncul real-time

**Cara 2: Filter**
1. Pilih **Kategori** (dropdown)
2. Pilih **Proyek** (dropdown)
3. Pilih **Date Range** (tanggal upload)
4. Daftar akan ter-filter

### OCR (Optical Character Recognition)

**Ekstrak teks dari gambar/scan:**

1. Upload gambar invoice/kwitansi
2. Sistem otomatis mendeteksi teks
3. Klik **"View Extracted Text"**
4. Teks bisa di-copy atau di-edit
5. Otomatis isi form transaksi dari teks

## AI Assistant

### Chat dengan AI

1. Klik **"AI Assistant"** di sidebar (atau tekan **Ctrl+K**)
2. Ketik pertanyaan, misal:
   - *"Berapa total pengeluaran bulan ini?"*
   - *"Proyek mana yang paling boros?"*
   - *"Buatkan laporan cashflow Januari-Maret"*
3. AI akan menjawab dengan data real-time
4. Jika diminta laporan, klik **"Generate Report"**

### Fitur AI

- **Data Analysis:** Analisis trend keuangan

## FAQ

### 1. **Bagaimana cara mengubah password?**

1. Klik **Profile** (pojok kanan atas)
2. Pilih **"Security Settings"**
3. Klik **"Change Password"**
4. Masukkan password lama dan password baru
5. Klik **"Update"**

---

### 2. **Kenapa saya tidak bisa menghapus transaksi?**

**Kemungkinan:**
- Anda tidak punya akses (hanya Admin/Manager yang bisa delete)
- Transaksi sudah di-lock karena sudah masuk laporan final

**Solusi:** Hubungi Admin untuk approval

---

### 3. **Dokumen saya gagal upload, kenapa?**

**Penyebab umum:**
- File terlalu besar (max 50MB per file)
- Format tidak didukung
- Koneksi internet terputus

**Solusi:**
- Compress file (gunakan ZIP)
- Convert ke PDF jika format tidak didukung
- Coba lagi dengan koneksi yang lebih stabil

---

### 4. **Bagaimana cara invite anggota tim?**

**Hanya Admin yang bisa invite:**
1. Klik **"Users"** di sidebar
2. Klik **"+ Invite User"**
3. Masukkan email
4. Pilih role (Admin/Manager/User)
5. Klik **"Send Invitation"**
6. Mereka akan terima email invite

---

### 5. **Data saya hilang, apakah bisa dikembalikan?**

**Ya!** Sistem otomatis backup setiap hari.

**Cara restore:**
1. Hubungi Admin
2. Sebutkan data yang hilang (nama proyek, tanggal transaksi, dll)
3. Admin akan restore dari backup (maksimal 30 hari terakhir)

---

### 6. **Aplikasi lambat/error, apa yang harus dilakukan?**

**Langkah troubleshooting:**

1. **Refresh halaman:** Tekan **Ctrl+R** (Windows) atau **Cmd+R** (Mac)
2. **Clear cache:** 
   - Chrome: **Ctrl+Shift+Delete** ‚Üí Clear browsing data
   - Pilih "Cached images and files" ‚Üí Clear
3. **Coba browser lain:** Firefox/Edge/Safari
4. **Cek koneksi internet:** Pastikan stabil

**Jika masih error:**
- Screenshot error message
- Kirim ke admin@natacare.com
- Sertakan: Browser, OS, langkah yang dilakukan

---

### 7. **Notifikasi terlalu banyak, bagaimana matikan?**

1. Klik **Profile** ‚Üí **"Notification Settings"**
2. Uncheck notifikasi yang tidak diinginkan:
   - Email notifications
   - Browser push notifications
   - In-app notifications
3. Klik **"Save"**

---

### 8. **Bagaimana cara export semua data?**

1. Klik **"Reports"** ‚Üí **"Data Export"**
2. Pilih data yang ingin di-export:
   - Projects
   - Transactions
   - Documents metadata
3. Pilih format: **Excel** atau **CSV**
4. Klik **"Export All Data"**
5. File akan di-generate (bisa memakan waktu jika data banyak)
6. Download setelah selesai

---
